# OSAKA

A static single-page website built as a hands-on practice project to explore and apply Bootstrap components and responsive design techniques.

## 🎯 Purpose

This project was built with the goal of practicing front-end development skills, specifically learning how to use Bootstrap's core components and grid system in a real project setting.

## 🛠️ Built With

- HTML5
- CSS3
- Bootstrap

## ✨ Bootstrap Components Used

- Navbar — responsive navigation bar
- Carousel — image slideshow
- Cards — content display blocks
- Grid System — responsive layout structure

## 📌 What I Learned

- How to structure a responsive single-page layout using Bootstrap's grid
- How to implement and customize Bootstrap components
- How to combine custom CSS with Bootstrap for styling
